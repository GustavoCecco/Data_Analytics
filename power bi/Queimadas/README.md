# Análise de Dados - Queimadas no Brasil

## Projeto

Este projeto tem como objetivo monitorar e analisar os focos de queimadas no Brasil, utilizando dados históricos e comparativos entre os anos de 2023 e 2024. O painel desenvolvido no Power BI permite a visualização de tendências, variações regionais e temporais, e fornece insights para ações preventivas e corretivas.

Os dados foram integrados e processados utilizando Python, e o painel foi construído para proporcionar uma análise detalhada e intuitiva das informações.

## Base de Dados

A base de dados foi extraída do site [INPE](https://terrabrasilis.dpi.inpe.br/queimadas/portal/). O INPE (Instituto Nacional de Pesquisas Espaciais) é uma instituição de pesquisa brasileira vinculada ao Ministério da Ciência, Tecnologia e Inovação (MCTI). Ele é amplamente reconhecido por suas contribuições nas áreas de monitoramento ambiental, meteorologia, sensoriamento remoto, ciência espacial e tecnologia aeroespacial. O INPE é essencial para a preservação ambiental no Brasil e para estudos climáticos e espaciais. Seus dados são usados por órgãos governamentais, universidades, ONGs e instituições internacionais para monitorar e mitigar impactos ambientais, como desmatamento e mudanças climáticas.

O monitoramento de queimadas pelo INPE é amplamente utilizado para alertar sobre picos de incêndios florestais e orientar políticas de controle e fiscalização.

## Funcionalidades

### Painéis interativos no Power BI:

- Comparação dos focos de queimadas entre 2023 e 2024.
- Análise regional (Norte, Centro-Oeste, Nordeste, Sudeste e Sul).
- Tendências mensais e variações percentuais.

### Processamento de dados com Python:

- Combinação de vários arquivos CSV em um único arquivo integrado.
- Preparação dos dados para uso no Power BI.
- Insights destacados:

### Destaques
Identificação das regiões e meses mais críticos.
Variação percentual para análise de aumento ou redução de queimadas.

## Processamento de Dados

Os dados originais estavam divididos em vários arquivos CSV. Foi necessário combinar todos os arquivos em um único arquivo que está no repositório para facilitar a análise no Power BI.
